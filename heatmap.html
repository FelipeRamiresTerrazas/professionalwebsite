how can I export the notebook i'm working on?

You can export a notebook from Snowsight in a few ways:

  1. Download as .ipynb: In the notebook's three-dot menu (⋮) at the top, select Download to save it as a .ipynb file locally.

  2. From the Workspace file browser: Right-click the .ipynb file in the left sidebar and choose Download.

These options let you export the notebook for use in Jupyter, VS Code, or other compatible environments.
